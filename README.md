# IELTS Reading Library

A complete, installable IELTS Reading PWA by **Exir Language Academy**.

- 447 readings
- B1, B2 and gentle C1
- Offline after the first complete visit
- Installable on iPhone, iPad, Android and desktop
- Timer, vocabulary, US pronunciation, highlights, questions and progress tracking

## Public address

After GitHub Pages is enabled:

**https://aminhosseini7.github.io/ielts-reading-library/**

## Publish from the GitHub website

1. Create a public repository named exactly `ielts-reading-library`.
2. Extract the GitHub-ready ZIP.
3. Upload all extracted files to the root of the repository.
4. Open **Settings → Pages**.
5. Choose **Deploy from a branch**.
6. Select `main` and `/ (root)`.
7. Click **Save**.

Students only open the public link. They do not need GitHub, ZIP files or technical knowledge.

## Updating the application

Replace the repository files with a newer build and commit the changes. GitHub Pages republishes the site, and the installed PWA checks for updates automatically.
